# STACK

This folder contains stack data structure examples in C++.

## What is included

- Stack operations such as push, pop, and top.
- Examples of stack-based algorithms and expression handling.
- Practice problems using last-in-first-out behavior.

## How to use

1. Open a file in this folder.
2. Study how values are pushed and popped.
3. Compile and run with:
   ```bash
   g++ filename.cpp -o stack
   ./stack
   ```

## Detailed explanation

A stack stores elements in last-in-first-out (LIFO) order. The last element added is the first one removed.

Key stack ideas:

- Push adds an element to the top.
- Pop removes the top element.
- Top reads the current top element without removing it.
- Stacks are useful for expression evaluation, recursion simulation, and backtracking.

Understanding stacks helps with parsing, undo operations, and managing nested structures.
