# subarray

This folder contains subarray and sliding window problems in C++.

## What is included

- Problems focused on contiguous ranges of an array.
- Examples of sum, maximum subarray, and range-based logic.
- Practice with sliding window and prefix sum techniques.

## How to use

1. Open a file in this folder.
2. Study how contiguous segments are selected and evaluated.
3. Compile and run with:
   ```bash
   g++ filename.cpp -o subarray
   ./subarray
   ```

## Detailed explanation

A subarray is a contiguous portion of an array. Many problems ask for the best or valid subarray under a condition.

Key subarray ideas:

- Sliding window: maintain a moving range and update the answer efficiently.
- Prefix sums: compute range sums quickly.
- Two-pointer techniques for ordered or condition-based subarrays.
- Handling variable window sizes and boundary checks.

Subarray practice builds strong intuition for array ranges and efficient scanning techniques.
