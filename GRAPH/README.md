# GRAPH

This folder contains graph algorithms and traversal examples in C++.

## What is included

- Graph representation using adjacency list and matrix.
- Traversal examples for BFS and DFS.
- Weighted graph and shortest path examples.
- Files such as `adjlist.cpp`, `dfs.cpp`, `matrixbfstrav.cpp`, and `gfgmindistance.cpp`.

## How to use

1. Open a graph example file.
2. Study how the graph nodes and edges are stored.
3. Compile and run:
   ```bash
   g++ adjlist.cpp -o adjlist
   ./adjlist
   ```

## Detailed explanation

Graphs are powerful structures for representing relationships between objects. In graph problems:

- Nodes represent entities.
- Edges represent connections.
- BFS explores the graph level by level.
- DFS explores as deep as possible before backtracking.
- Weighted graphs include edge costs for shortest path computation.

Understanding graphs is critical for many real-world problems such as routing, scheduling, and network design.
