# STRING

This folder contains string manipulation and string algorithm examples in C++.

## What is included

- String search and processing problems.
- Files like `01.cpp`, `02.cpp`, `03.cpp`, and `08pandrom.cpp`.
- Examples of palindrome checks, comparisons, and formatting.

## How to use

1. Open a file in this folder.
2. Study how string functions and loops are used.
3. Compile and run with:
   ```bash
   g++ 01.cpp -o string
   ./string
   ```

## Detailed explanation

Strings are sequences of characters. String problems often involve searching, comparing, slicing, and modifying text.

Key string ideas:

- Traversing characters using loops.
- Comparing strings lexicographically.
- Detecting palindromes and repeated substrings.
- Using `std::string` methods for efficient operations.

String manipulation is essential for input parsing, algorithmic pattern matching, and text processing.
