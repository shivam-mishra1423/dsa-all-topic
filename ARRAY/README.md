# ARRAY

This folder contains array practice problems and algorithm examples in C++.

## What is included

- Basic array operations such as traversal, search, and modification.
- Problem files like `01.cpp`, `02.cpp`, `03.cpp`, and others.
- Examples of duplicates removal, lexicographic order generation, and matrix rotation.
- Specialized files like `gfgarraydublicate.cpp`, `LEXIOGRAPHI.cpp`, and `rotatearr90.cpp`.

## How to use

1. Open a `.cpp` file in this folder.
2. Study how the array is declared, used, and manipulated.
3. Compile and run with a C++ compiler:
   ```bash
   g++ 01.cpp -o 01
   ./01
   ```

## Detailed explanation

Arrays are one of the simplest and most important data structures. An array stores elements in a contiguous memory block, and each element can be accessed using an index. This folder focuses on core array techniques:

- traversal: visiting each element in order using loops.
- indexing: reading or writing values based on position.
- searching: locating elements either by scanning or by using more efficient methods when the array is sorted.
- duplicates: finding and handling repeated values.
- ordering: generating lexicographic order and manipulating array order.
- transformations: rotating rows or columns, working with 2D arrays.

Arrays are foundational for almost every other data structure. Practicing these examples helps you write faster, more correct code for larger problems in dynamic programming, hashing, and graph algorithms.
