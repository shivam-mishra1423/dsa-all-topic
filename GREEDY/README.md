# GREEDY

This folder contains greedy algorithm examples in C++.

## What is included

- Problems solved by making the best local decision at each step.
- Examples may include scheduling, selection, or incremental choice patterns.

## How to use

1. Open a file in this folder.
2. Analyze the local decision rule used in the algorithm.
3. Compile and run with:
   ```bash
   g++ filename.cpp -o filename
   ./filename
   ```

## Detailed explanation

Greedy algorithms build a solution step by step by choosing the best immediate option. In many cases, a local optimum leads to a globally optimal solution.

Key greedy ideas:

- Choose the best available option at each step.
- Do not revisit earlier choices.
- Use greedy strategy when problem properties guarantee correctness.
- Greedy methods are often efficient and easy to implement.

Practicing greedy problems helps you recognize when a simple choice strategy is enough and when a more complex method is required.
