# QUEUE

This folder contains queue data structure examples in C++.

## What is included

- Queue operations such as enqueue and dequeue.
- Examples of first-in-first-out behavior.
- Practice code to build and use queues.

## How to use

1. Open a file in this folder.
2. Study how items are added and removed.
3. Compile and run with:
   ```bash
   g++ filename.cpp -o queue
   ./queue
   ```

## Detailed explanation

A queue processes elements in a first-in-first-out (FIFO) order. The first element added is the first one removed.

Key queue ideas:

- Enqueue adds an element to the rear.
- Dequeue removes an element from the front.
- Queues are useful for breadth-first search, scheduling, and buffering.
- Implementations can use arrays, linked lists, or STL containers.

Understanding queues helps with problems that require ordered processing of tasks or events.
