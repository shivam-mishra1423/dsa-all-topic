# STL

This folder contains C++ Standard Template Library examples.

## What is included

- Examples of using STL containers such as `vector`, `stack`, `queue`, `map`, and `set`.
- Practice code showing how STL simplifies algorithm implementation.
- Files like `stl.cpp`, `stleg.cpp`, and others.

## How to use

1. Open a file in this folder.
2. Observe how STL containers replace manual data structure code.
3. Compile and run with:
   ```bash
   g++ stl.cpp -o stl
   ./stl
   ```

## Detailed explanation

The C++ Standard Template Library (STL) provides ready-made containers and algorithms.

Key STL ideas:

- Use `vector` for dynamic arrays.
- Use `stack` and `queue` for LIFO/FIFO behavior.
- Use `map` and `unordered_map` for associative lookup.
- Use `set` for ordered unique elements.
- Standard algorithms reduce manual implementation and improve reliability.

Practicing with STL is important for writing clean, efficient C++ code quickly.
