# BASIC PROGRAMMING

This folder contains basic programming examples and function practice in C++.

## What is included

- Fundamental programming constructs such as loops, conditions, and functions.
- Simple practice files organized under `FUNCTION` and other examples.
- Examples to build programming intuition before moving to advanced algorithms.

## How to use

1. Open a file in this folder.
2. Learn how variables, control flow, and functions are used.
3. Compile and run with:
   ```bash
   g++ filename.cpp -o filename
   ./filename
   ```

## Detailed explanation

Basic programming is the foundation of software development. This folder teaches:

- Variables and data types
- Input/output operations
- Conditional statements (`if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Functions and modular design
- Simple problem solving using code structure

A strong grasp of these basics is essential before tackling data structures, algorithms, or any real-world code challenge.
